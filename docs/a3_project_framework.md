# A3 项目框架

## 1. 项目定位

本项目是一个基于大模型的个性化学习资源生成与学习规划多智能体系统，面向高校学生的课程学习、期末复习和实习技能提升场景。

系统通过自然语言生成学生画像，结合课程资料、考试安排和学习目标，由多个智能体协作生成学习资源、练习题和学习计划，并将学习计划落地为可执行任务。

## 2. 核心模块

### 2.1 用户与权限模块

负责注册、登录、JWT 鉴权、当前用户获取、多用户数据隔离。

### 2.2 学生画像模块

负责根据用户描述生成学生画像，包括基础信息、学习目标、薄弱点、可学习时间、学习偏好。

### 2.3 学习信息解析模块

负责解析考试安排、课程资料、学习目标和知识点。

### 2.4 多智能体协作模块

包括 ProfileAgent、ParserAgent、ResourceAgent、QuizAgent、PlannerAgent、EvaluatorAgent。

### 2.5 学习资源生成模块

负责生成知识点讲解、复习提纲、练习题、编程案例、思维导图或脚本。

### 2.6 学习计划与任务模块

负责生成学习计划、预览确认、一键导入任务、任务状态跟踪、优先级管理。

### 2.7 记录、反馈与展示模块

负责 AI 操作日志、资源历史记录、任务完成统计、答题记录和前端展示。

## 3. 核心闭环

### 3.1 学生画像生成闭环

用户输入 → 学生画像生成 → Pydantic 校验 → student_profiles 入库

### 3.2 资源生成闭环

课程资料 → RAG 检索 → Agent 生成资源 → learning_resources 入库

### 3.3 学习计划与任务闭环

学习计划预览 → 用户确认 → tasks 入库 → 任务完成记录 → 学生画像更新
